# Планировщик
## Описание:
Программа для отслеживания задач пользователя
## Задание
Создать окно приложения с возможностью добавления задачи, удаления задачи, указания приоритета (низкий, средний или высокий) и установки дедлайна. Задачи будут отображаться в таблице, в которой показывается имя, приоритет, дедлайн и оставшееся время в днях до дедлайна.
